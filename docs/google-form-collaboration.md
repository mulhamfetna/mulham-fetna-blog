# نموذج طلب تعاون — جاهز للنسخ إلى Google Forms

مرجع لبناء نموذج «طلب تعاون ومقترح فعالية». انسخ كل عنوان سؤال وخياراته كما هي.
هذا الملف **ليس** صفحة منشورة على الموقع — هو دليل بناء النموذج فقط.

> **قبل أن تبدأ — إعدادات النموذج (Settings):**
> - **Collect email addresses → ON** (إلزامي: يلزم للرد التلقائي وللتواصل).
> - **Responses → Get email notifications for new responses → ON** (لتصلك الطلبات على بريدك).
> - **Presentation → Confirmation message:** الصق رسالة الرد التلقائي (في نهاية هذا الملف).
> - قسّم النموذج إلى **6 أقسام (Sections)** بالعناوين أدناه.

---

## عنوان النموذج (Form title)

```
نموذج طلب تعاون ومقترح فعالية
```

## وصف النموذج (Form description)

```
يسعدنا اهتمامكم بالتعاون مع ملهم فتنة. لضمان دراسة طلبكم وتوافقه مع الجدول والبيئة الاحترافية المطلوبة، يُرجى تعبئة كافة الحقول. هذا النموذج مخصّص للمحاضرات التفاعلية وورش العمل القصيرة فقط؛ أما الحقائب التدريبية المتكاملة فلها مسار تنسيقي وتعاقدي مختلف. للأسئلة أو إرفاق ملف تعريف جهتكم: contact@mulhamfetna.com
```

## القسم الأول: معلومات الجهة الطالبة للتعاون

**1. اسم المؤسسة / المبادرة / الجهة المنظمة**
`Short answer` · **Required**

**2. نوع الجهة**
`Dropdown` · **Required**
- أكاديمية (جامعة / كلية / مدرسة)
- تطوعية / مبادرة شبابية
- شركة خاصة
- منظمة غير ربحية
- جهة حكومية
- أخرى

**3. اسم الشخص المسؤول عن التنسيق المباشر**
`Short answer` · **Required**

**4. الصفة الوظيفية للمنسّق**
`Short answer` · **Required**

**5. البريد الإلكتروني للمنسّق**
`Short answer` · **Required** · *(فعّل Response validation → Text → Email)*
> ملاحظة: إذا فعّلت "Collect email addresses" في الإعدادات فسيُجمع بريد المُرسِل تلقائياً؛ أبقِ هذا الحقل ليصلك بريد **المنسّق** تحديداً.

**6. رقم التواصل (واتساب مع رمز الدولة)**
`Short answer` · **Required**
> نص مساعد (Description): مثال: ‎+963 9xx xxx xxx

**7. رابط الموقع الإلكتروني أو حسابات التواصل الاجتماعي للجهة**
`Short answer` · *(اختياري)*

## رسالة التأكيد بعد الإرسال (Confirmation message)

> Settings → Presentation → Confirmation message — الصق:

```
شكراً لتواصلكم. تم استلام مقترحكم وسنراجعه بعناية لمطابقته مع الجدول والمعايير المطلوبة. سنعود إليكم عبر البريد الإلكتروني الذي زوّدتمونا به. نعتذر سلفاً عن عدم الرد على المقترحات غير المكتملة، أو ما يتجاوز المحاضرات وورش العمل.
```

## بعد إنشاء النموذج

1. **Send → <> (Embed HTML):** انسخ الرابط. الـ id هو المقطع الطويل بين `/d/e/` و `/viewform`.
2. أرسل هذا الـ id لأضعه في صفحة `/work-with-me/invite/` (يحلّ محل `REPLACE_WITH_GOOGLE_FORM_ID`).
3. **Responses → ⋮ → Get email notifications for new responses → ON.**
